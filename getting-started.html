Getting Started

Getting started with nickel.rs is extremely easy. This guide will walk you through the steps necessary to write your first nickel app.

1. Get a recent version of Rust and Cargo

In order to write a nickel app you first need to make sure to have a recent version of Rust and Cargo. In case you are wondering Cargo is the package manager for Rust. It makes dependency and build management a breeze. The easiest way to install both is just a simple one liner on your shell.

$ curl -s http://www.rust-lang.org/rustup.sh | sudo sh

2. Create a new working directory and cd into it

$ mkdir nickel-demo
$ cd nickel-demo

3. Create a Cargo.toml file

The Cargo.toml file is the manifest that describes all dependencies of your app and also tells Cargo which file is the entry point of your app. Just place it directly into the nickel-demo directory.

[package]

name = "nickel-demo"
version = "0.1.0"
authors = [ "your-name@gmail.com" ]

[[bin]]

name = "nickel-demo"
path = "src/main.rs"

[dependencies.nickel]

git = "https://github.com/nickel-org/nickel.rs.git"

4. Create a main.rs file

You may have spotted the path = "src/main.rs" in your Cargo.tomlfile. This is where we will put the code for your first nickel app. You may want to choose a different file name but at the time of writing Cargo assumes your file to be in src/main.rs for the cargo run command to work which we will use in a moment.

Paste this into your main.rs file in the src directory. 

extern crate nickel;

use std::io::net::ip::Ipv4Addr;
use nickel::{ Nickel, Request, Response };

fn main() {
    let mut server = Nickel::new();
    
    fn a_handler (_request: &Request, response: &mut Response) { 
        response.send("hello world"); 
    }

    server.get("/hello", a_handler);
    server.listen(Ipv4Addr(127, 0, 0, 1), 6767);
}

5. Run your app

Congrats! You created your first nickel app. Just type cargo run on your console and visit http://localhost:6767/hello in your browser. It should print out hello world.
